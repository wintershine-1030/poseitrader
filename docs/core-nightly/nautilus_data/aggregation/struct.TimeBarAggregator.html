Struct TimeBarAggregator

pub struct TimeBarAggregator<H>
where
    H: FnMut(Bar),
{ /* private fields */ }
Expand description

Provides a means of building time bars aggregated from quote and trades.

At each aggregation time interval, a bar is created and sent to the handler.
