Struct Price

Source 
#[repr(C)]
pub struct Price {
    pub raw: PriceRaw,
    pub precision: u8,
}
Expand description

Represents a price in a market.

The number of decimal places may vary. For certain asset classes, prices may have negative values. For example, prices for options instruments can be negative under certain conditions.

Handles up to {FIXED_PRECISION} decimals of precision.

  • PRICE_MAX = {PRICE_MAX}
  • PRICE_MIN = {PRICE_MIN}

Fields§

§raw: PriceRaw

Represents the raw fixed-point value, with precision defining the number of decimal places.

§precision: u8

The number of decimal places, with a maximum of {FIXED_PRECISION}.
